### QList - Métodos y Operaciones

`QList` es el contenedor secuencial más utilizado en Qt. En **Qt 6**, `QList` y `QVector` se han unificado: `QList` ahora almacena datos **contiguos en memoria** (como `std::vector`). En **Qt 5**, `QList` usaba un array de punteros a los elementos si estos eran mayores que un puntero.

#### Evolución de QList en Qt 5 vs Qt 6

| Aspecto | Qt 5 `QList` | Qt 6 `QList` |
| :--- | :--- | :--- |
| **Almacenamiento** | Array de punteros (para tipos > sizeof(void*)) | Contiguo en memoria (como std::vector) |
| **Relación con QVector** | Clases separadas | `QVector` es alias de `QList` |
| **Rendimiento** | Fragmentación posible | Óptimo para caché |
| **Recomendación** | Usar QVector para datos contiguos | Usar QList para todo |

#### Creación e Inicialización

```cpp
#include <QList>
#include <QDebug>

void creacion() {
    // Lista vacía
    QList<int> l1;
    
    // Con lista de inicialización
    QList<int> l2 = {1, 2, 3, 4, 5};
    
    // Con tamaño y valor (Qt 6)
    QList<QString> l3(3, "hola");  // {"hola", "hola", "hola"}
    
    qDebug() << l2 << l3;
}
```

#### `swapItemsAt()` - Intercambiar Elementos

```cpp
void intercambiar() {
    QList<int> lista = {10, 20, 30, 40, 50};
    
    // Intercambiar posiciones 1 y 3
    lista.swapItemsAt(1, 3);  // {10, 40, 30, 20, 50}
    
    qDebug() << lista;
}
```

#### `takeAt()`, `takeFirst()`, `takeLast()` - Extraer Elementos

A diferencia de `removeAt()`, las funciones `take*` devuelven el elemento eliminado.

```cpp
void extraer() {
    QList<QString> lista = {"uno", "dos", "tres", "cuatro"};
    
    // takeAt() - extraer por posición
    QString segundo = lista.takeAt(1);  // "dos", lista = {"uno", "tres", "cuatro"}
    
    // takeFirst() - extraer el primero
    QString primero = lista.takeFirst(); // "uno", lista = {"tres", "cuatro"}
    
    // takeLast() - extraer el último
    QString ultimo = lista.takeLast();   // "cuatro", lista = {"tres"}
    
    qDebug() << segundo << primero << ultimo << lista;
}
```

#### `move()` - Mover Elemento de Posición

```cpp
void moverElemento() {
    QList<QString> lista = {"A", "B", "C", "D", "E"};
    
    // Mover el elemento de la posición 0 a la posición 3
    lista.move(0, 3);  // {"B", "C", "D", "A", "E"}
    
    qDebug() << lista;
}
```

#### `replaceAt()` y Operaciones de Modificación

```cpp
void modificar() {
    QList<int> lista = {10, 20, 30, 40, 50};
    
    // replace() - reemplazar en posición
    lista.replace(2, 99);     // {10, 20, 99, 40, 50}
    
    // Operador += para concatenar
    QList<int> otra = {60, 70};
    lista += otra;             // {10, 20, 99, 40, 50, 60, 70}
    
    // Operador << para agregar
    lista << 80 << 90;        // {10, 20, 99, 40, 50, 60, 70, 80, 90}
    
    qDebug() << lista;
}
```

#### `sliced()` - Sublista (Qt 6.0+)

```cpp
void sublista() {
    QList<int> lista = {10, 20, 30, 40, 50, 60};
    
    // sliced(posición, longitud)
    QList<int> sub = lista.sliced(2, 3);  // {30, 40, 50}
    
    // sliced(posición) - desde posición hasta el final
    QList<int> resto = lista.sliced(4);   // {50, 60}
    
    // mid() - equivalente en Qt 5 (también disponible en Qt 6)
    QList<int> medio = lista.mid(1, 2);   // {20, 30}
    
    qDebug() << sub << resto << medio;
}
```

#### Búsqueda y Filtrado

```cpp
void busquedaFiltrado() {
    QList<int> lista = {5, 10, 15, 20, 25, 10};
    
    // contains()
    bool tiene15 = lista.contains(15);         // true
    
    // indexOf() / lastIndexOf()
    int primera = lista.indexOf(10);           // 1
    int ultima = lista.lastIndexOf(10);        // 5
    
    // count()
    int veces = lista.count(10);               // 2
    
    // startsWith() / endsWith()
    bool empieza = lista.startsWith(5);        // true
    
    qDebug() << tiene15 << primera << ultima << veces;
}
```

#### Conversión a/desde Contenedores Estándar

```cpp
void conversiones() {
```

### Conclusión

`QList` es el contenedor más versátil de Qt. En Qt 6, al unificarse con `QVector`, combina la API rica de ambos con un rendimiento óptimo de memoria contigua. Métodos como `takeAt()`, `swapItemsAt()`, `move()` y el operador `<<` hacen que las operaciones comunes sean más expresivas que con `std::vector`.
