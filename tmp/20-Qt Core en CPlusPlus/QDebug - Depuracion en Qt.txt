### QDebug - Depuración en Qt

`QDebug` es el sistema de logging y depuración de Qt. Proporciona una interfaz fluida y type-safe para imprimir mensajes de depuración, similar a `std::cout` pero con integración nativa con todos los tipos de Qt y niveles de severidad configurables.

#### Niveles de Mensajes

Qt define cuatro niveles de severidad, cada uno con su propia función:

```cpp
#include <QDebug>

void niveles() {
    qDebug() << "Mensaje de depuración";     // Información general
    qInfo() << "Mensaje informativo";        // Información importante
    qWarning() << "Mensaje de advertencia";  // Posible problema
    qCritical() << "Mensaje crítico";        // Error grave
    // qFatal("Error fatal: %s", "descripción"); // Termina la aplicación
}
```

| Función | Severidad | Uso |
| :--- | :--- | :--- |
| `qDebug()` | Baja | Trazas de desarrollo |
| `qInfo()` | Media | Información de estado |
| `qWarning()` | Alta | Problemas potenciales |
| `qCritical()` | Muy alta | Errores graves |
| `qFatal()` | Fatal | Termina el programa |

#### Impresión de Tipos Qt

`qDebug()` sabe imprimir nativamente los tipos de Qt sin conversiones manuales.

```cpp
#include <QDebug>
#include <QString>
#include <QList>
#include <QMap>
#include <QRect>
#include <QPoint>

void tiposQt() {
    QString texto = "Hola";
    QList<int> lista = {1, 2, 3};
    QMap<QString, int> mapa;
    mapa["a"] = 1;
    mapa["b"] = 2;
    QRect rect(10, 20, 100, 50);
    QPoint punto(5, 10);
    
    qDebug() << texto;   // "Hola"
    qDebug() << lista;   // (1, 2, 3)
    qDebug() << mapa;    // QMap(("a", 1)("b", 2))
    qDebug() << rect;    // QRect(10,20 100x50)
    qDebug() << punto;   // QPoint(5,10)
}
```

#### Formato con `qDebug()` estilo printf

```cpp
void formatoPrintf() {
    int edad = 30;
    const char *nombre = "Carlos";
    
    // Estilo printf (sin tipo seguro)
    qDebug("Nombre: %s, Edad: %d", nombre, edad);
    
    // Estilo stream (type-safe, recomendado)
    qDebug() << "Nombre:" << nombre << "Edad:" << edad;
}
```

#### Categorías de Logging

Las categorías permiten filtrar mensajes de log por módulo o componente.

```cpp
#include <QDebug>
#include <QLoggingCategory>

// Declarar categoría
Q_LOGGING_CATEGORY(lcRed, "app.red")
Q_LOGGING_CATEGORY(lcDB, "app.database")

void categorias() {
    // Mensajes categorizados
    qCDebug(lcRed) << "Conexión establecida";
    qCWarning(lcDB) << "Consulta lenta detectada";
    qCInfo(lcRed) << "Datos recibidos:" << 1024 << "bytes";
}
```

Para filtrar categorías en tiempo de ejecución:
```cpp
void configurarFiltros() {
    // Desactivar mensajes de debug de la categoría "app.red"
    QLoggingCategory::setFilterRules("app.red.debug=false");
    
    // Activar solo warnings de "app.database"
    QLoggingCategory::setFilterRules("app.database.debug=false\n"
                                     "app.database.info=false");
}
```

#### Personalizar la Salida de Log

```cpp
#include <QDebug>
#include <QDateTime>

// Handler personalizado
void miHandler(QtMsgType tipo, const QMessageLogContext &ctx, const QString &msg) {
    QString timestamp = QDateTime::currentDateTime().toString("hh:mm:ss.zzz");
    QString nivel;
    
    switch (tipo) {
    case QtDebugMsg:    nivel = "DEBUG"; break;
    case QtInfoMsg:     nivel = "INFO"; break;
    case QtWarningMsg:  nivel = "WARN"; break;
    case QtCriticalMsg: nivel = "ERROR"; break;
    case QtFatalMsg:    nivel = "FATAL"; break;
    }
    
    fprintf(stderr, "[%s] %s: %s (%s:%u)\n",
            qPrintable(timestamp),
            qPrintable(nivel),
            qPrintable(msg),
            ctx.file, ctx.line);
}

// Instalar el handler
void instalarHandler() {
    qInstallMessageHandler(miHandler);
    
    qDebug() << "Este mensaje usa el formato personalizado";
}
```

#### Hacer que un Tipo Propio sea Imprimible con `qDebug()`

```cpp
#include <QDebug>

struct Punto3D {
    double x, y, z;
};

// Sobrecargar operator<< para QDebug
QDebug operator<<(QDebug dbg, const Punto3D &p) {
    QDebugStateSaver saver(dbg);  // Restaura el estado al final
    dbg.nospace() << "Punto3D(" << p.x << ", " << p.y << ", " << p.z << ")";
    return dbg;
}

void imprimirPersonalizado() {
    Punto3D p{1.0, 2.5, 3.7};
    qDebug() << "Mi punto:" << p;
    // Salida: Mi punto: Punto3D(1, 2.5, 3.7)
}
```

### Conclusión

`QDebug` ofrece un sistema de logging completo y flexible que supera a `std::cout` en usabilidad. Las categorías de logging, los niveles de severidad, la impresión nativa de tipos Qt y la capacidad de personalizar el handler hacen de `QDebug` una herramienta indispensable para la depuración y el monitoreo de aplicaciones Qt.
