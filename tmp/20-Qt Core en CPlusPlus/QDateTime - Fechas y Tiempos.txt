### QDateTime - Fechas y Tiempos

Qt proporciona tres clases principales para manejar fechas y tiempos: `QDate` (solo fecha), `QTime` (solo hora) y `QDateTime` (fecha + hora). Ofrecen una API más intuitiva que `<chrono>` de C++ para operaciones comunes como formateo, parsing, zonas horarias y aritmética de fechas.

#### QDate - Solo Fecha

```cpp
#include <QDate>
#include <QDebug>

void fechas() {
    // Fecha actual
    QDate hoy = QDate::currentDate();
    qDebug() << "Hoy:" << hoy;                    // 2024-12-25
    
    // Fecha específica
    QDate navidad(2024, 12, 25);
    
    // Componentes
    qDebug() << "Año:" << navidad.year();          // 2024
    qDebug() << "Mes:" << navidad.month();         // 12
    qDebug() << "Día:" << navidad.day();           // 25
    qDebug() << "Día de la semana:" << navidad.dayOfWeek();  // 3 (miércoles)
    qDebug() << "Día del año:" << navidad.dayOfYear();       // 360
    
    // Validación
    bool valida = QDate::isValid(2024, 2, 29);     // true (año bisiesto)
    bool invalida = QDate::isValid(2023, 2, 29);   // false
    
    qDebug() << "Bisiesto 2024:" << valida;
}
```

#### QTime - Solo Hora

```cpp
#include <QTime>
#include <QDebug>

void tiempos() {
    // Hora actual
    QTime ahora = QTime::currentTime();
    qDebug() << "Ahora:" << ahora;
    
    // Hora específica
    QTime alarma(7, 30, 0);  // 07:30:00
    
    // Componentes
    qDebug() << "Hora:" << alarma.hour();        // 7
    qDebug() << "Minutos:" << alarma.minute();   // 30
    qDebug() << "Segundos:" << alarma.second();  // 0
    
    // Diferencia en segundos
    QTime inicio(9, 0, 0);
    QTime fin(17, 30, 0);
    int segundos = inicio.secsTo(fin);
    qDebug() << "Horas de trabajo:" << segundos / 3600.0; // 8.5
    
    // Medir tiempo transcurrido (uso simple)
    QTime crono;
    crono.start();
    // ... operación costosa ...
    qDebug() << "Tardó:" << crono.elapsed() << "ms";
}
```

#### QDateTime - Fecha y Hora Combinadas

```cpp
#include <QDateTime>
#include <QDebug>

void fechaHora() {
    // Fecha y hora actual
    QDateTime ahora = QDateTime::currentDateTime();
    qDebug() << "Ahora:" << ahora;
    
    // Con zona horaria UTC
    QDateTime ahoraUTC = QDateTime::currentDateTimeUtc();
    
    // Desde componentes
    QDateTime especifico(QDate(2024, 6, 15), QTime(14, 30, 0));
    
    // Desde timestamp Unix
    QDateTime desdeEpoch = QDateTime::fromSecsSinceEpoch(1700000000);
    qDebug() << "Desde epoch:" << desdeEpoch;
    
    // A timestamp Unix
    qint64 epoch = ahora.toSecsSinceEpoch();
    qDebug() << "Epoch actual:" << epoch;
}
```

#### Formateo con `toString()`

```cpp
void formateo() {
    QDateTime ahora = QDateTime::currentDateTime();
    
    // Formatos predefinidos
    qDebug() << ahora.toString(Qt::ISODate);        // "2024-12-25T14:30:00"
    qDebug() << ahora.toString(Qt::RFC2822Date);     // "Wed, 25 Dec 2024 14:30:00"
    
    // Formato personalizado
    qDebug() << ahora.toString("dd/MM/yyyy");        // "25/12/2024"
    qDebug() << ahora.toString("hh:mm:ss");          // "14:30:00"
    qDebug() << ahora.toString("dddd, d 'de' MMMM 'de' yyyy");
    // "miércoles, 25 de diciembre de 2024"
    
    // Tabla de tokens:
    //   d    = día sin cero (1-31)
    //   dd   = día con cero (01-31)
    //   ddd  = abreviatura día (Lun)
    //   dddd = nombre completo (Lunes)
    //   M    = mes sin cero (1-12)
    //   MM   = mes con cero (01-12)
    //   MMM  = abreviatura mes (Dic)
    //   MMMM = nombre completo (Diciembre)
    //   yy   = año 2 dígitos (24)
    //   yyyy = año 4 dígitos (2024)
    //   h/hh = hora 12h
    //   H/HH = hora 24h
    //   m/mm = minutos
    //   s/ss = segundos
    //   AP   = AM/PM
}
```

#### Parsing con `fromString()`

```cpp
void parsing() {
    // Parsear desde formato ISO
    QDateTime dt1 = QDateTime::fromString("2024-12-25T14:30:00", Qt::ISODate);
    
    // Parsear desde formato personalizado
    QDateTime dt2 = QDateTime::fromString("25/12/2024 14:30", "dd/MM/yyyy hh:mm");
    
    // Parsear solo fecha
    QDate fecha = QDate::fromString("2024-12-25", "yyyy-MM-dd");
    
    // Verificar si el parsing fue exitoso
    if (dt1.isValid()) {
        qDebug() << "Parseado:" << dt1;
    } else {
        qDebug() << "Formato inválido";
    }
}
```

#### Aritmética de Fechas

```cpp
void aritmetica() {
    QDate hoy = QDate::currentDate();
    
    // Sumar/restar días
    QDate manana = hoy.addDays(1);
    QDate ayer = hoy.addDays(-1);
    
    // Sumar/restar meses y años
    QDate siguienteMes = hoy.addMonths(1);
    QDate siguienteAnio = hoy.addYears(1);
    
    // Diferencia entre fechas
    int diasHastaNavidad = hoy.daysTo(QDate(hoy.year(), 12, 25));
    
    QDateTime ahora = QDateTime::currentDateTime();
    
    // Sumar horas, minutos, segundos
    QDateTime enUnaHora = ahora.addSecs(3600);
    QDateTime enUnDia = ahora.addDays(1);
    
    // Diferencia en segundos
    qint64 diffSegs = ahora.secsTo(enUnaHora);
    
    qDebug() << "Mañana:" << manana;
    qDebug() << "Días hasta Navidad:" << diasHastaNavidad;
    qDebug() << "Diferencia:" << diffSegs << "segundos";
}
```

#### Zonas Horarias

```cpp
#include <QTimeZone>

void zonasHorarias() {
    QDateTime ahora = QDateTime::currentDateTime();
    
    // Convertir a UTC
    QDateTime utc = ahora.toUTC();
    qDebug() << "Local:" << ahora;
    qDebug() << "UTC:" << utc;
    
    // Zona horaria específica
    QTimeZone madrid("Europe/Madrid");
    QTimeZone tokyo("Asia/Tokyo");
    
    QDateTime enMadrid = ahora.toTimeZone(madrid);
    QDateTime enTokyo = ahora.toTimeZone(tokyo);
    
    qDebug() << "Madrid:" << enMadrid;
    qDebug() << "Tokyo:" << enTokyo;
    
    // Listar zonas horarias disponibles
    QList<QByteArray> zonas = QTimeZone::availableTimeZoneIds();
    qDebug() << "Total zonas:" << zonas.size();
}
```

#### Comparación con `<chrono>` (C++11/20)

| Operación | Qt | `<chrono>` |
| :--- | :--- | :--- |
| **Fecha actual** | `QDate::currentDate()` | Necesita conversión manual |
| **Formateo** | `toString("dd/MM/yyyy")` | `std::format` (C++20) |
| **Parsing** | `fromString(str, fmt)` | `std::chrono::parse` (C++20) |
| **Aritmética** | `addDays()`, `addMonths()` | `year_month_day` + `days{}` |
| **Zonas horarias** | `QTimeZone` | `std::chrono::zoned_time` (C++20) |
| **Simplicidad** | Alta | Baja (antes de C++20) |

### Conclusión

`QDate`, `QTime` y `QDateTime` ofrecen una API completa e intuitiva para el manejo de fechas y tiempos. El formateo personalizado con `toString()`, el parsing con `fromString()`, la aritmética de fechas y el soporte de zonas horarias con `QTimeZone` cubren la mayoría de necesidades sin la complejidad de `<chrono>` previo a C++20.
