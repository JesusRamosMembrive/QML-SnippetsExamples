### QString - Métodos y Operaciones

`QString` es la clase de Qt para el manejo de cadenas de texto. A diferencia de `std::string`, `QString` almacena internamente texto en **UTF-16**, lo que le permite manejar caracteres Unicode de forma nativa. Además, utiliza **Implicit Sharing (Copy-on-Write)**, haciendo que las copias sean extremadamente baratas.

#### Creación y Conversión

```cpp
#include <QString>
#include <QDebug>

void creacion() {
    // Desde literal
    QString s1 = "Hola Mundo";
    
    // Desde número
    QString s2 = QString::number(42);        // "42"
    QString s3 = QString::number(3.14, 'f', 2); // "3.14"
    
    // Desde std::string
    std::string std_str = "texto estándar";
    QString s4 = QString::fromStdString(std_str);
    
    // De vuelta a std::string
    std::string de_vuelta = s4.toStdString();
    
    // Desde UTF-8 explícito
    QString s5 = QString::fromUtf8("日本語");
    
    qDebug() << s1 << s2 << s3 << s4 << s5;
}
```

#### `arg()` - Formateo de Cadenas

`arg()` es el reemplazo de Qt para `printf`. Es type-safe y soporta encadenamiento.

```cpp
void formateo() {
    QString nombre = "Carlos";
    int edad = 30;
    double salario = 2500.50;
    
    // Sustitución posicional
    QString msg = QString("Nombre: %1, Edad: %2, Salario: %3€")
                    .arg(nombre)
                    .arg(edad)
                    .arg(salario, 0, 'f', 2);
    // Resultado: "Nombre: Carlos, Edad: 30, Salario: 2500.50€"
    
    // Padding con anchura fija
    QString num = QString("%1").arg(42, 5, 10, QChar('0'));
    // Resultado: "00042"
    
    qDebug() << msg << num;
}
```

#### Búsqueda y Comprobación

```cpp
void busqueda() {
    QString texto = "Qt es un framework potente para C++";
    
    // contains() - busca subcadena
    bool tiene = texto.contains("framework");           // true
    bool tieneSinCase = texto.contains("FRAMEWORK", Qt::CaseInsensitive); // true
    
    // startsWith() / endsWith()
    bool empieza = texto.startsWith("Qt");              // true
    bool termina = texto.endsWith("C++");               // true
    
    // indexOf() - posición de la primera ocurrencia
    int pos = texto.indexOf("framework");               // 12
    
    // count() - número de ocurrencias
    int veces = texto.count("e");                       // 3
    
    // isEmpty() / isNull()
    QString vacia = "";
    QString nula;
    bool esVacia = vacia.isEmpty();  // true
    bool esNula = nula.isNull();     // true
    // Nota: isNull() es true solo si no se ha asignado ningún valor
    
    qDebug() << tiene << pos << veces;
}
```

#### Manipulación de Texto

```cpp
void manipulacion() {
    // trimmed() - elimina espacios al inicio y final
    QString s1 = "  Hola Mundo  ";
    QString limpio = s1.trimmed();     // "Hola Mundo"
    
    // simplified() - trimmed() + reduce espacios internos múltiples
    QString s2 = "  Hola    Mundo  ";
    QString simple = s2.simplified();  // "Hola Mundo"
    
    // toUpper() / toLower()
    QString upper = QString("hola").toUpper();  // "HOLA"
    QString lower = QString("HOLA").toLower();  // "hola"
    
    // replace()
    QString s3 = "Hola Mundo";
    s3.replace("Mundo", "Qt");         // "Hola Qt"
    
    // remove()
    QString s4 = "abc123def";
    s4.remove(QRegularExpression("[0-9]")); // "abcdef"
    
    // left(), right(), mid() - extraer subcadenas
    QString s5 = "abcdefgh";
    QString izq = s5.left(3);         // "abc"
    QString der = s5.right(3);        // "fgh"
    QString medio = s5.mid(2, 4);     // "cdef"
    
    // chopped() - quita N caracteres del final (Qt 5.10+)
    QString s6 = "archivo.txt";
    QString sinExt = s6.chopped(4);   // "archivo"
    
    qDebug() << limpio << simple << upper << izq << medio;
}
```

#### Conversión a Tipos Numéricos

```cpp
void conversiones() {
    // A entero
    QString s1 = "42";
    int num = s1.toInt();               // 42
    
    // Con verificación de éxito
    bool ok;
    int num2 = QString("abc").toInt(&ok); // ok = false, num2 = 0
    
    // A double
    double d = QString("3.14").toDouble();  // 3.14
    
    // A long long
    qint64 grande = QString("9999999999").toLongLong();
    
    // Desde hexadecimal
    int hex = QString("FF").toInt(&ok, 16);  // 255
    
    qDebug() << num << d << grande << hex;
}
```

#### `split()` y `join()`

```cpp
void splitJoin() {
    // split() - dividir cadena
    QString csv = "uno,dos,tres,cuatro";
    QStringList partes = csv.split(",");
    // partes = ["uno", "dos", "tres", "cuatro"]
    
    // split() con opciones
    QString conVacios = "a,,b,,c";
    QStringList sinVacios = conVacios.split(",", Qt::SkipEmptyParts);
    // sinVacios = ["a", "b", "c"]
    
    // join() - unir lista en cadena (ver QStringList)
    QString unida = partes.join(" - ");
    // unida = "uno - dos - tres - cuatro"
    
    qDebug() << partes << unida;
}
```

#### Comparación entre `QString` y `std::string`

| Característica | `QString` | `std::string` |
| :--- | :--- | :--- |
| **Codificación** | UTF-16 | Sin codificación definida |
| **Implicit Sharing** | Sí (COW) | No |
| **Formateo** | `arg()` type-safe | `printf` o `std::format` (C++20) |
| **Regex** | `QRegularExpression` integrado | `std::regex` (más lento) |
| **Conversión numérica** | `toInt()`, `toDouble()` | `std::stoi()`, `std::stod()` |
| **Unicode** | Nativo | Requiere bibliotecas externas |

### Conclusión

`QString` es una de las clases más utilizadas en Qt. Su soporte nativo para Unicode, las operaciones de manipulación seguras y eficientes, y la compartición implícita la convierten en una alternativa superior a `std::string` para aplicaciones que requieren internacionalización o manipulación avanzada de texto.
