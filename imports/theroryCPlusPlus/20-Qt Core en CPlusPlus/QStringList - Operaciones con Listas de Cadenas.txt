### QStringList - Operaciones con Listas de Cadenas

`QStringList` es una especialización de `QList<QString>` que añade métodos específicos para manipular listas de cadenas de texto. Es uno de los tipos más usados en Qt para procesar datos textuales como rutas de archivos, líneas de configuración o resultados de parsing.

#### Creación

```cpp
#include <QStringList>
#include <QDebug>

void creacion() {
    // Desde lista de inicialización
    QStringList frutas = {"manzana", "pera", "naranja", "plátano"};
    
    // Con operador <<
    QStringList colores;
    colores << "rojo" << "verde" << "azul";
    
    // Desde QString::split()
    QString csv = "uno,dos,tres";
    QStringList partes = csv.split(",");
    
    qDebug() << frutas << colores << partes;
}
```

#### `filter()` - Filtrar por Patrón

```cpp
void filtrar() {
    QStringList archivos = {"main.cpp", "utils.h", "config.xml", "app.cpp", "style.css"};
    
    // Filtrar por subcadena
    QStringList cpp = archivos.filter(".cpp");
    // cpp = {"main.cpp", "app.cpp"}
    
    // Filtrar con expresión regular
    QStringList headers = archivos.filter(QRegularExpression("\\.h$"));
    // headers = {"utils.h"}
    
    // Filtrar case-insensitive
    QStringList lista = {"Hola", "hola", "HOLA", "mundo"};
    QStringList holas = lista.filter("hola", Qt::CaseInsensitive);
    // holas = {"Hola", "hola", "HOLA"}
    
    qDebug() << cpp << headers << holas;
}
```

#### `join()` - Unir en una Sola Cadena

```cpp
void unir() {
    QStringList partes = {"usr", "local", "bin"};
    
    // join() con separador
    QString ruta = partes.join("/");        // "usr/local/bin"
    QString csv = partes.join(", ");        // "usr, local, bin"
    QString pegado = partes.join("");       // "usrlocalbin"
    
    qDebug() << ruta << csv;
}
```

#### `replaceInStrings()` - Reemplazar en Todas las Cadenas

```cpp
void reemplazar() {
    QStringList archivos = {"file1.txt", "file2.txt", "file3.txt"};
    
    // Reemplazar subcadena en todos los elementos
    archivos.replaceInStrings(".txt", ".csv");
    // archivos = {"file1.csv", "file2.csv", "file3.csv"}
    
    // Con expresión regular
    QStringList nombres = {"Juan García", "Ana López", "Pedro Ruiz"};
    nombres.replaceInStrings(QRegularExpression("\\s.*"), "");
    // nombres = {"Juan", "Ana", "Pedro"} (elimina desde el espacio)
    
    qDebug() << archivos << nombres;
}
```

#### `sort()` y `removeDuplicates()`

```cpp
void ordenarYLimpiar() {
    QStringList lista = {"banana", "apple", "cherry", "apple", "banana"};
    
    // sort() - ordenar alfabéticamente
    lista.sort();
    // lista = {"apple", "apple", "banana", "banana", "cherry"}
    
    // sort() case-insensitive
    lista.sort(Qt::CaseInsensitive);
    
    // removeDuplicates() - eliminar duplicados (devuelve número de eliminados)
    int eliminados = lista.removeDuplicates();
    // lista = {"apple", "banana", "cherry"}, eliminados = 2
    
    qDebug() << lista << "eliminados:" << eliminados;
}
```

#### `contains()` y Búsqueda

```cpp
void busqueda() {
    QStringList lenguajes = {"C++", "Python", "Java", "Rust", "Go"};
    
    // contains() - ¿existe la cadena?
    bool tiene = lenguajes.contains("Python");              // true
    bool tieneSinCase = lenguajes.contains("python", Qt::CaseInsensitive); // true
    
    // indexOf()
    int pos = lenguajes.indexOf("Java");                    // 2
    
    // Buscar con regex
    int posRegex = lenguajes.indexOf(QRegularExpression("^R.*")); // 3 (Rust)
    
    qDebug() << tiene << pos << posRegex;
}
```

#### Caso Práctico: Procesar Argumentos de Línea de Comandos

```cpp
#include <QCoreApplication>
#include <QStringList>
#include <QDebug>

void procesarArgumentos() {
    QStringList args = QCoreApplication::arguments();
    
    // El primer argumento es el nombre del programa
    QString programa = args.first();
    
    // Filtrar flags (empiezan con --)
    QStringList flags = args.filter(QRegularExpression("^--"));
    
    // Verificar si existe un flag específico
    bool verbose = args.contains("--verbose");
    
    qDebug() << "Programa:" << programa;
    qDebug() << "Flags:" << flags;
    qDebug() << "Verbose:" << verbose;
}
```

### Conclusión

`QStringList` extiende `QList<QString>` con operaciones especializadas como `filter()`, `join()`, `replaceInStrings()` y `removeDuplicates()` que simplifican enormemente el procesamiento de texto. Es especialmente útil para parsear archivos, manejar rutas y procesar datos CSV o de configuración.
