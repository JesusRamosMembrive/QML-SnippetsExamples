### Operadores en C++

En C++, los operadores son símbolos especiales que se utilizan para realizar operaciones en operandos. Hay varios tipos de operadores, incluyendo aritméticos, lógicos y relacionales, que permiten realizar cálculos, comparaciones y evaluaciones lógicas.

#### 1. **Operadores Aritméticos**

Los operadores aritméticos se utilizan para realizar operaciones matemáticas básicas.

| Operador | Nombre         | Ejemplo      | Descripción                          |
|----------|----------------|--------------|--------------------------------------|
| `+`      | Suma           | `a + b`      | Suma de `a` y `b`                    |
| `-`      | Resta          | `a - b`      | Resta de `b` de `a`                  |
| `*`      | Multiplicación | `a * b`      | Producto de `a` y `b`                |
| `/`      | División       | `a / b`      | Cociente de `a` y `b`                |
| `%`      | Módulo         | `a % b`      | Resto de la división de `a` entre `b`|

##### Ejemplo de Operadores Aritméticos

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int b = 3;

    cout << "Suma: " << a + b << endl;           // 13
    cout << "Resta: " << a - b << endl;          // 7
    cout << "Multiplicación: " << a * b << endl; // 30
    cout << "División: " << a / b << endl;       // 3
    cout << "Módulo: " << a % b << endl;         // 1

    return 0;
}
```

#### 2. **Operadores Lógicos**

Los operadores lógicos se utilizan para realizar operaciones lógicas, generalmente en expresiones booleanas.

| Operador | Nombre   | Ejemplo       | Descripción                            |
|----------|----------|---------------|----------------------------------------|
| `&&`     | AND      | `a && b`      | Verdadero si tanto `a` como `b` son verdaderos |
| `||`     | OR       | `a || b`      | Verdadero si al menos uno de `a` o `b` es verdadero |
| `!`      | NOT      | `!a`          | Verdadero si `a` es falso               |

##### Ejemplo de Operadores Lógicos

```cpp
#include <iostream>
using namespace std;

int main() {
    bool a = true;
    bool b = false;

    cout << "AND: " << (a && b) << endl; // 0 (false)
    cout << "OR: " << (a || b) << endl;  // 1 (true)
    cout << "NOT: " << (!a) << endl;     // 0 (false)

    return 0;
}
```

#### 3. **Operadores Relacionales**

Los operadores relacionales se utilizan para comparar dos valores. El resultado de una operación relacional es un valor booleano (`true` o `false`).

| Operador | Nombre           | Ejemplo   | Descripción                          |
|----------|------------------|-----------|--------------------------------------|
| `==`     | Igual a          | `a == b`  | Verdadero si `a` es igual a `b`      |
| `!=`     | No igual a       | `a != b`  | Verdadero si `a` no es igual a `b`   |
| `>`      | Mayor que        | `a > b`   | Verdadero si `a` es mayor que `b`    |
| `<`      | Menor que        | `a < b`   | Verdadero si `a` es menor que `b`    |
| `>=`     | Mayor o igual que| `a >= b`  | Verdadero si `a` es mayor o igual que `b` |
| `<=`     | Menor o igual que| `a <= b`  | Verdadero si `a` es menor o igual que `b` |

##### Ejemplo de Operadores Relacionales

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int b = 20;

    cout << "Igual a: " << (a == b) << endl;         // 0 (false)
    cout << "No igual a: " << (a != b) << endl;      // 1 (true)
    cout << "Mayor que: " << (a > b) << endl;        // 0 (false)
    cout << "Menor que: " << (a < b) << endl;        // 1 (true)
    cout << "Mayor o igual que: " << (a >= b) << endl; // 0 (false)
    cout << "Menor o igual que: " << (a <= b) << endl; // 1 (true)

    return 0;
}
```

### Uso Combinado de Operadores

En muchos casos, los operadores aritméticos, lógicos y relacionales se combinan para construir expresiones más complejas.

##### Ejemplo de Uso Combinado

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 5;
    int y = 10;
    int z = 15;

    // Combinación de operadores aritméticos y relacionales
    bool resultado1 = (x + y) > z; // true

    // Combinación de operadores lógicos y relacionales
    bool resultado2 = (x < y) && (y < z); // true

    cout << "Resultado 1: " << resultado1 << endl; // 1 (true)
    cout << "Resultado 2: " << resultado2 << endl; // 1 (true)

    return 0;
}
```

#### 4. **Operadores de Asignación**

| Operador | Nombre | Ejemplo | Equivalente |
|----------|--------|---------|-------------|
| `=` | Asignación | `a = b` | — |
| `+=` | Suma y asigna | `a += b` | `a = a + b` |
| `-=` | Resta y asigna | `a -= b` | `a = a - b` |
| `*=` | Multiplica y asigna | `a *= b` | `a = a * b` |
| `/=` | Divide y asigna | `a /= b` | `a = a / b` |
| `%=` | Módulo y asigna | `a %= b` | `a = a % b` |

#### 5. **Operadores de Incremento y Decremento**

| Operador | Nombre | Ejemplo | Descripción |
|----------|--------|---------|-------------|
| `++` | Incremento | `a++` o `++a` | Incrementa `a` en 1 |
| `--` | Decremento | `a--` o `--a` | Decrementa `a` en 1 |

- **Prefijo (`++a`):** Incrementa y devuelve el valor incrementado.
- **Postfijo (`a++`):** Devuelve el valor actual y luego incrementa.

```cpp
int a = 5;
int b = ++a; // a=6, b=6 (incrementa primero)
int c = a++; // a=7, c=6 (devuelve primero, luego incrementa)
```

#### 6. **Operador Ternario**

El operador ternario `? :` es una forma compacta de un `if-else`:

```cpp
int edad = 18;
string resultado = (edad >= 18) ? "Mayor de edad" : "Menor de edad";
```

#### 7. **Operadores de Bits (Bitwise)**

Operan sobre la representación binaria de los operandos:

| Operador | Nombre | Ejemplo | Descripción |
|----------|--------|---------|-------------|
| `&` | AND | `a & b` | AND bit a bit |
| `\|` | OR | `a \| b` | OR bit a bit |
| `^` | XOR | `a ^ b` | XOR bit a bit |
| `~` | NOT | `~a` | Complemento a uno |
| `<<` | Shift izq. | `a << n` | Desplaza bits a la izquierda |
| `>>` | Shift der. | `a >> n` | Desplaza bits a la derecha |

```cpp
int a = 0b1010; // 10 en binario
int b = 0b1100; // 12 en binario

int andResult = a & b;  // 0b1000 = 8
int orResult  = a | b;  // 0b1110 = 14
int xorResult = a ^ b;  // 0b0110 = 6
int notResult = ~a;     // Complemento a uno
int shiftL    = a << 1; // 0b10100 = 20
int shiftR    = a >> 1; // 0b0101 = 5
```

#### 8. **Operador Spaceship `<=>` (C++20)**

El operador de comparación triple devuelve el resultado completo de una comparación:

```cpp
#include <compare>

int a = 5, b = 10;
auto resultado = a <=> b; // std::strong_ordering::less

if (resultado < 0)  cout << "a < b" << endl;
if (resultado == 0) cout << "a == b" << endl;
if (resultado > 0)  cout << "a > b" << endl;
```

### Operadores en el Contexto de Qt

#### Operadores de flags con enums Qt

Qt usa extensivamente operadores bitwise para combinar flags:

```cpp
Qt::WindowFlags flags = Qt::Window | Qt::FramelessWindowHint | Qt::WindowStaysOnTopHint;

Qt::Alignment alineacion = Qt::AlignLeft | Qt::AlignTop;

// Verificar si un flag está activo
if (flags & Qt::FramelessWindowHint) {
    qDebug() << "Ventana sin marco";
}
```

#### `operator<<` para QDebug

Qt sobrecarga `operator<<` para imprimir cualquier tipo registrado con `QDebug`:

```cpp
#include <QDebug>
#include <QPoint>
#include <QRect>

QPoint punto(10, 20);
QRect rect(0, 0, 100, 50);
qDebug() << "Punto:" << punto;   // QPoint(10,20)
qDebug() << "Rect:" << rect;     // QRect(0,0 100x50)
```

#### Funciones de utilidad Qt (alternativas a operadores)

```cpp
#include <QtGlobal>

int valor = 150;
int acotado = qBound(0, valor, 100);  // 100 (acota entre 0 y 100)
int minimo = qMin(10, 20);            // 10
int maximo = qMax(10, 20);            // 20
qreal absoluto = qAbs(-3.14);         // 3.14
```
